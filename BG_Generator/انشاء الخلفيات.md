كيفية التشغيل:
ضع جميع الملفات الثلاثة في مجلد جديد باسم BG_Generator.
تأكد من تثبيت المكتبات المطلوبة داخل المجلد:
code
Bash
npm install puppeteer fs-extra
افتح ملف config.json واختر نوع التأثير من الـ 10 المتاحة في ملف الـ HTML (مثال: squares, waves, dots, hexagons).
قم بتشغيل البرنامج:
code
Bash
node generator.js
لماذا هذا الكود أفضل؟
Loopable: جميع التأثيرات تعود لنقطة البداية في نهاية المدة (Duration).
Dynamic: يمكنك تغيير اللون الأساسي والشفافية بضغطة زر.
Modern: يستخدم تقنيات GSAP للحصول على حركة سلسة جداً (Smooth) تشبه فيديوهات الموشن جرافيك الاحترافية.
